# FormulaPath (数式導出ゲーム)

`FormulaPath` は、数式の成り立ちをクイズ感覚で楽しみながら理解を深めることができる、iOS向けの数式導出ゲームアプリです。

ほんのりピンクとパープルのグラデーションを基調としたモダンで親しみやすいUIデザインを採用し、中学生レベルの「解の公式の導出」から、大学レベルの高度な数式まで、ステップバイステップで導出プロセスを学ぶことができます。

---

## 本プロジェクトのこだわりと技術的特徴

### 1. 疎結合・カプセル化を意識したアーキテクチャ
アプリ全体の設計において、表示ロジックと描画エンジンを完全に分離する「疎結合」を強く意識しています。
* **画面遷移の一括管理**: ホーム画面（`FormulaPathHomeView`）では、遷移先を `FormulaPathDestination`（Enum）としてカプセル化し、`NavigationStack` の `navigationDestination` を用いて一括管理しています。
* **レンダラーの抽象化**: 数式描画は `LaTeXView` としてコンポーネント化されているため、将来的に描画ライブラリを切り替える必要が生じた場合でも、各画面のコードを変更することなく、本コンポーネントの内部ロジックを修正するだけで対応が可能です。

### 2. WKWebView × KaTeX による美しい数式表示
標準のテキストコンポーネントでは表示が崩れやすい分数や積分記号（$\int$）などの複雑な数式を、Web技術で最速を誇る数式レンダリングライブラリ「KaTeX」を用いて美しく描画します。
* **自動スタイル適応**: 独自のCSS調整により、iOSのシステム設定（ライトモード・ダークモード）を検知し、数式の文字色を自動的に美しく反転させる仕様となっています。
* **誤操作の防止**: ユーザーが意図せず数式をスクロールしたり、ピンチイン・アウトで拡大縮小したりできないよう、WebView側のインタラクションを最適化しています。

### 3. 数式対応の共通選択肢ボタン (`ChoiceButton`)
クイズの回答用ボタンには、再利用性の高い `ChoiceButton` を使用しています。
* **心地よい操作性**: ボタンをタップした際に「ぷにっとした」バネ感のあるフィードバックが得られるよう、独自の `ButtonStyle`（`spring` アニメーション）を適用しています。
* **数式文字への対応**: 選択肢内のテキストにも `LaTeXView` を内包させることで、ボタン内の文字も完全に数式フォーマットで美しくレンダリングできる柔軟な設計となっています。

## 開発者向けの注意事項

### 1. バックスラッシュのエスケープ処理について

Swiftのコード内でLaTeXソースを記述する際は、バックスラッシュ（`\`）のエスケープが課題となります。

* 画面側（`HighStageSelectionView` など）では、Swiftの拡張文字列リテラル（Raw String: `Format #"\int..."#`）を活用することで、エスケープなしで直感的に数式コードを記述できるようにしています。
* `LaTeXView` 内部では、JavaScript側へ値を安全に渡すために、内部的にバックスラッシュを適切に置換・調整（`\\\\` へのエスケープ処理）を行っています。

### 2. ファイル名・フォルダ名の整合性について

iOSの実機環境においては、ファイルシステムの大文字・小文字が厳密に区別されます。コード内のリソース指定文字列とディスク上の実際のファイルパス（例: `LaTeXVeiw.swift` のタイポや、今後導入予定のローカルアセットフォルダ名など）に齟齬が生じないよう、命名には十分ご注意ください。
